= RNA-seq data analysis
:experimental:

Create a directory dedicated to the analyses:

[source,cmd]
----
mkdir analysis
----

And move into it:

[source,cmd]
----
cd analysis
----

Load the environment:

[source,cmd]
----
. ~/rnaseq/.rnaseqenv
----
